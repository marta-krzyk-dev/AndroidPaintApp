# AndroidPaintApp
A simple Android paint app made with:
- Java, 
- Eclipse, 
- Android 4.2.2. (API 11).

## Features:
- 12 colour palette,
- 3 brush sizes,
- "New picture" button,
- "Save" button saving pictures to phone's gallery.
